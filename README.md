# Violeta Guinea - Portfolio CV

Web profesional de marca personal desarrollada con HTML, CSS y JavaScript.

## Estructura del proyecto

```text
.
├── assets/
│   └── Fotocurriculum.png
├── src/
│   ├── main.js
│   ├── styles.css
│   └── pages/
│       ├── about.html
│       ├── contacto.html
│       ├── experiencia.html
│       ├── formacion.html
│       └── skills.html
├── CNAME
├── index.html
└── README.md
```

## Descripción

- `index.html`: portada principal del portfolio.
- `src/pages/`: páginas interiores de la web.
- `src/styles.css`: estilos globales del proyecto.
- `src/main.js`: funcionalidades JavaScript comunes.
- `assets/`: imágenes utilizadas por la web.
- `CNAME`: configuración del dominio personalizado para GitHub Pages.

## Despliegue

El proyecto está preparado para publicarse con GitHub Pages manteniendo `index.html` y `CNAME` en la raíz del repositorio.
